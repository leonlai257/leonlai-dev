# React three Boilerplate

## Introduction

This boilerplate could be used for building websites with 3d graphic elements, using react three and related libraries

## Technical Stack

3D Rendering: React 3 fiber, React 3 drei
routing: wouter

## How to use

### Installation

```zsh
yarn install
# or
npm install
```

### Run locally

```zsh
yarn dev
# or
npm run dev
```
